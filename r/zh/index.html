Radare 是一个可移植的「逆向」框架，其功能有：

  • 反汇编（汇编）很多不同的架构
  • 配合本地原生的或远程的调试器，进行调试（gdb，rap，webui，r2pipe，winedbg 和 windbg）
  • 支持 Linux，*BSD，Windows，OSX，Android，iOS，Solaris 和 Haiku
  • 执行文件系统取证和数据雕刻
  • 使用 Python，Javascript，Go 和更多的语言，编写脚本
  • 支持协作分析，通过内置的 web 服务器
  • 可视化很多文件类型的数据结构
  • 给程序打补丁，以发现新特性或修复缺陷
  • 使用强大的分析能力，加速逆向处理
  • 援助软件爆破
